4) 9 点最终结论

01

5月末市场特征

三速度格局 — 头部科技成长 / 资源最快，中坚小盘次之，大盘价值最弱。机构 + 外资偏好科技创新与新能源。牛市中后期，需防短期回调。

02

下半年主线

科技革新 + 高端制造为核心；辅以周期资源、国企改革 / 基建链、消费恢复；金融防御稳健底仓。成长与价值共振，风格轮动加剧。

03

5-8 个重点题材

AI / 算力、半导体国产化、新能源车 + 锂电、光伏风电、资源周期、机器人工业制造、医疗创新、消费升级。政策 + 需求 + 业绩三重支撑。

04

ETF 配置方向

主题 ETF + 宽基。科技主题（科创 50 / 创业板 / 半导体 / AI / 机器人）+ 新能源（新能车 / 光伏 / 储能）；宽基平滑波动；红利对冲。

05

持有期限建议

新能源车、核心芯片 = 中长期；军工 / 游戏 = 短线；红利 / 金融 = 组合核心常备。涨幅巨大且基本面改进有限的板块 = 谨慎。

06

避免追高

估值极高且上涨主因资金推高的板块严格回避：短线"网红"股、部分周期股、商业模式不清晰的标的。不重仓泡沫。

07

下半年组合建议

核心-卫星：宽基 + 红利 / 防御为核心，科技 / 新能源 / 医疗为卫星。宏观风险加大时提高防御权重；风险偏好回升时增配成长。

08

主要风险因素

消费 / 房市持续疲软；海外资本退潮 / 美元急升；地缘冲突加剧；监管收紧。科技板块内部风险（供给过剩 / 创新失败）和流动性边际骤降。

09

首要跟踪指标

宏观：PMI / 社融 / M2 / 出口。市场：上证 + 创业板趋势 / 成交额 / 融资余额 / ETF 资金流。题材：AI 算力订单 / 国产替代进度 / 新能源车销量 / 铜油价格。

成长 vs 价值+73.1 pp

高股息 vs 成长−
高股息相对失效。利率走低环境下吸引力减弱，资金流向成长。仅适合防御与轮动转换。
动量因子强
科技股动量持续。上涨持续吸引跟风资金，动量因子显著有效。
估值水平↑
成长板块估值水平上移，价值板块估值处历史低位。
风险偏好修复中
指数与科技主题 ETF 普遍上涨；防御型（公用事业）相对落后；进入风险偏好修复阶段。

2025 GDP 增速 ~5%；高技术制造业 PMI 连续维持扩张；出口在全球复苏 + 人民币贬值推动下保持增长。消费与科技驱动稳健，传统周期（地产、基建）复苏力度较弱。
政策环境双线政策
财政货币偏宽松；地方债 / 专项债支持基建；地产调控边际放松；科技自主政策（芯片、半导体、AI、绿色能源）加码 + 补贴。对新兴行业有力扶持，对过剩产能 / 高房价继续施压。
流动性 / 资金面宽松延续
央行适度宽松，利率中枢下行。ETF 资金流入明显（科创综指 ETF 已吸引数百亿）。北向年内净流入偏低，但科技股回调时仍有逢低买入。融资余额维持高位。
海外因素利率见顶
美联储转鸽 + 全球利率见顶 → 风险偏好回暖；美元指数回落 → 资金回流 A 股；全球 AI 加速落地为半导体 / 计算机带来外部共振；大宗商品调整对传统资源利空、对新能源 / 出口制造利好。
板块驱动总结： 科技 / 新基建受益于政策 + 全球趋势； 金融受益于利率 + 监管环境； 地产 / 原材料受限于政策收紧 + 行业调整； 消费需看内需恢复 + 居民预期。

CHAPTER 10 · FINAL VIEW
01

过去一年板块格局

成长 + 科技板块领跑（创业板 +100.9%、科创 50 +89.3%）；大盘蓝筹 + 传统周期股相对落后；产业结构升级清晰可见（电子市值超传统银行业，地产腰斩）。

02

强势 / 弱势行业

强势 = 电子、通信、新能源车、半导体、生物医药、军工、光伏储能；弱势 = 房地产、建筑装饰、基础化工、煤炭 / 石油石化、部分资源、传统消费。

03

风格轮动判断

市场明显由防御（高股息、传统金融）转向进攻（科技成长）。高股息策略与价值股暂时失效；动量 + 主题类 ETF 表现较好。

04

ETF / 基金配置

核心 = 宽基 ETF（沪深 300、中证 500、科创 50）+ 行业龙头 ETF；卫星 = 新能源、科技主题 ETF（AI、半导体、新能源车、军工）；防御转换可持银行 / 消费 / 高股息。

05

未来 3–12 月建议

继续看好科技创新 + 内需驱动（高配科技、通信、新能源车；中性配置消费金融；低配地产 / 资源）。短期警惕高估值板块回调风险。
